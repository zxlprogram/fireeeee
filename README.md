# Fireeeee - 火災疏散模擬器

## 最新更新 (2026-07-03)

### 煙霧擴散邏輯改進

最新版本對煙霧擴散機制進行了重大優化：

- **不再疊加擴散**: 移除了煙霧層級的重複疊加，使擴散更加合理
- **擴散範圍增加**: 煙霧擴散範圍增加一格，提高環境危險感知
- **濃度計算函數**: 改用 `sin(x+π/2)^0.68+1` 函數來計算濃度增長，使濃度變化更加平滑自然
- **smartEscape 感測器邏輯改進**: 當感測器判斷出濃煙危險時，不再絕對不經過該路徑，改以權重方式判斷，增加路徑選擇的靈活性
- **用戶移動 Bug 修正**: 修正了使用者必須在每一個 tick 移動的 bug，提升使用體驗

## 項目介紹

這是一個使用 Java 開發的火災疏散模擬器，用於模擬建築物內人員在火災情況下的疏散行為和安全評估。

## 技術棧

- **語言**: Java 100%

## 功能特性

- 火災蔓延模擬
- 煙霧擴散計算
- 人員逃生路徑規劃
- 多種逃生策略模擬
- 實時模擬可視化

## 開發狀態

持續開發中，最近進行了煙霧擴散引擎的重大改進。
