模擬方式:
在同樣沒有消防措施的情況下，模擬有無系統的存活率與存活時間，我們以這套系統中，透過"使用者會知道哪些資訊"影響使用者在每一步中的決策，以模擬系統的效果

模擬缺陷:
沒有消防車、滅火器，火勢固定蔓延

量化方式:
我們另存活率為生還人數/總人數。
我們另逃出生天的人的存活時長=房屋從剛起火完全燒毀的時長，被燒死的則是從房屋起火到人死了的時長；我們比較【一個人在有系統跟沒系統的存活時長差距】的平均值。

量化結果:
經過計算與驗證，這個系統相對沒有系統時的優化程度並非線性，而是固定的常數時間差，時間差與火勢蔓延的速率成正比，比率為5.734；沒有系統的逃生率約1.8%，有系統的約15.6%
